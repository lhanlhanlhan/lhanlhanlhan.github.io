骑行：罗庄西里 <-> 首钢园区
2023-06-10 20:34, 北京

其实真正能治愈你的不是人类，而是睡觉、美食，可爱的小动物，以及抬头就可见的风景。

最近北京天气深得我心：天是蓝的，云是低的，太阳是辣的，空气是甜的。周末放着这种天气不出去溜达，感觉就像花两万块钱点一桌子菜不去吃一样，浪费。

路线

去程：知春路 -> 万柳 -> 西四环 (南向) -> 杏石口路 -> 永定河引水渠 -> 苹果园 -> 首钢
回程：首钢 -> 苹果园 -> 田村 -> 西四环 (北向) -> 北三环西路 -> 知春路
总路程 来回约 42km。
费时 考虑到等红绿灯及爬坡，单程大约花费 1.5 – 2h。
坡度 海淀 -> 石景山方向预计 爬升 80m；有 2 个坡度较大的桥（长春桥、西黄村桥）。
夜骑 永引渠段照明不好，杏石口路/田村路某些路段有较多大车。
耗能 共约消耗能量 1,150 kCal。

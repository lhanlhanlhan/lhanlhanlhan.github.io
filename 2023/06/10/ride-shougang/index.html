骑行：罗庄西里 <-> 首钢园区
2023-06-10 08:34, 北京

最近北京天气深得我心：天是蓝的，云是白的，太阳是辣的，空气是甜的。 前段时间在心中积累的各种 Shit，这几天就好像都被天上挂着的那轮火球晒得蒸发掉了似的。

路线

去程：罗庄 -> 万柳 -> 西四环 (南向) -> 杏石口路 -> 永定河引水渠 -> 苹果园 -> 首钢
回程：首钢 -> 苹果园 -> 田村 -> 西四环 (北向) -> 北三环西路 -> 罗庄
单程 约 20km
费时 考虑到等红绿灯及爬坡，单程大约花费 1.5 – 2h。
坡度 海淀 -> 石景山方向预计 爬升 80m；有 2 个坡度较大的桥（长春桥、西黄村桥）。
夜骑 永引渠段照明不好，杏石口路/田村路某些路段有较多大车。
耗能 共约消耗能量 1,150 kCal。
